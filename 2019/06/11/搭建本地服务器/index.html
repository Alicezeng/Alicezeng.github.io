搭建本地服务器

发表于 2019-06-11 | 阅读数 次

方法一：使用http-server

准备工作：下载node.js，按照默认选项安装到指定文件夹
1.打开CMD，在选好的文件夹下全局安装服务器：npm install http-server -g
2.安装完成后，在此文件夹目录下，输入命令：http-server，启动服务器
3.启动好服务器后，直接将待调试的文件放到该目录下即可

方法二：使用http

通过node server.js启动服务器后，可访问地址为http://localhost:9000/billShow.html（前提是根目录下有public文件，public文件里包括了billShow.html）

方法三 使用express()

` server.js
var path=require(‘path’);
var express=require(‘express’);
var app=express();
app.get(‘/billShow’,function(req,res){
// res.send(‘123’)
res.sendFile(path.join(__dirname,’/public/billShow.html’))
});
app.listen(5000,function(err){
if(err){
console.log(err)
}
console.log(‘服务器已启动’)
})
通过node server.js启动服务器后，可访问地址为http://localhost:5000/billShow（前提是根目录下有public文件，public文件里包括了billShow.html）
